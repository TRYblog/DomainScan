# DomainScan
一个简单的子域名爆破工具
by:Try
# 说明
原理很简单,采用多线程爆破子域名,可自定义dns查询服务器，默认:8.8.8.8
(目前还未解决泛解析的问题，还是有点bug)
# 参数
```
  -domain string
        一级域名,如:baidu.com
  -f string
        加载字典
  -s int
        线程 (default 5)
  -server string
        dns服务器 (default "8.8.8.8")
```
